# golife
Conway's Game of Life created in golang.

It can run on the terminal.

<img src="https://github.com/mitsutoshi/wiki/raw/master/golife_demo.png" width="480px">

## Key Operations

* **Space**: Stop or Start.
* **Ctrl + R**: Reset cells
* **Ctrl + U**: Speed up
* **Ctrl + D**: Speed down
* **Esc**: Exit
